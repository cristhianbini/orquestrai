# Base de Conhecimento OrquestraAI

Fabrica automatizada de sistemas. Escalavel ate 180+ IAs.

## Core
| Diretorio | Uso |
|---|---|
| metas/ | Metas estrategicas (a IA trabalha ate alcançar) |
| pendencias/ | Sprint corrente |
| licoes/ | Aprendizados imortais |
| decisoes/ | ADRs assinados pelo humano |
| blocos/ | Blocos LAVE catalogados |

## IAs e Agentes
| Diretorio | Uso |
|---|---|
| agentes/ | Conhecimento por papel (planner,researcher,critic,executor,auditor,voice) |
| gateways/ | Configuração de provedores (OpenAI,Anthropic,Google,Ollama,Zhipu,etc) |
| avaliacoes/ | Score de qualidade por IA |
| custo/ | Rastreamento de custo por provedor/modelo |

## Automação
| Diretorio | Uso |
|---|---|
| skills/ | Habilidades importaveis (GitHub) ou criadas |
| skills/importadas/ | Skills baixadas de repos externos |
| skills/criadas/ | Skills criadas no projeto |
| skills/testadas/ | Skills validadas e prontas |
| workflows/ | Definicoes de workflow (sequencial,paralelo,react) |
| loops/ | Loops autonomos que trabalham ate atingir meta |
| loops/ativos/ | Loops rodando agora |
| loops/pausados/ | Loops pausados pelo humano |
| loops/concluidos/ | Loops que atingiram o objetivo |
| templates/ | Templates de projeto prontos |
