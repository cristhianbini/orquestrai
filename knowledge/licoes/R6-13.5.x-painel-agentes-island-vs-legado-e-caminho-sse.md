## Os 3 bugs corrigidos (causa raiz)
### Bug 1 — borda "done" invisivel
AgentCard.spotlight so tratava running/error; 'done' caia no else ('border-line'
= igual a idle). Card terminava sem estado visual. FIX: branch
`status === 'done' -> border-green + shadow`.

### Bug 2 (CAUSA RAIZ REAL) — nenhuma animacao durante a run
useMasRun buscava `/api/mas/last` UMA vez no mount. O painel React fica montado,
mas a run e disparada pelo CHAT LEGADO -- o React nunca era avisado da run nova.
Resultado: ficava preso no runId do mount; SSE so pegava runs antigas (ja
concluidas = so replay = so 'done'). PROVA: no mesmo disparo, REACT_SSE_DEBUG
mostrava run_id diferente do B162_DEBUG legado. FIX: polling de /api/mas/last a
cada 2s, troca runId quando muda. (Gambiarra funcional -- ver follow-up.)

### Bug 3 — brilho fraco
Keyframe agent-pulse-cyan oscilava 14->30px em cor ~45% alpha, imperceptivel.
FIX: glow ate 40px, alpha 95% no pico, + brilho interno; etiqueta maior (11px,
bold) com animacao por estado + badge-flash (2x) no 'done'.
